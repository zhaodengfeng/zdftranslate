# Chrome Web Store Description (Chrome 应用商店说明)

## Short Description (短描述)
Simple and efficient bilingual translation. Supports OpenAI, OpenRouter, Custom API, DeepL, and YouTube subtitles.
简单高效的双语对照翻译插件，支持 OpenRouter、自定义 API、OpenAI、DeepL 及 YouTube 双语字幕。

## Long Description (长描述)

**[English]**

ZDFTranslate is a powerful browser extension designed for reading foreign language content efficiently. It provides a seamless bilingual reading experience by displaying translations side-by-side with the original text.

**Key Features:**

*   **Bilingual Display:** Smartly inserts translations next to paragraphs, preserving the original layout.
*   **Multiple Translation Engines:**
    *   **OpenRouter (New):** Access hundreds of top-tier models like GPT-4o, Claude 3.5, Gemini Pro, Llama 3 via a single key.
    *   **Custom Services (New):** Connect to any OpenAI-compatible (LocalAI, vLLM) or Anthropic-compatible API endpoint.
    *   **Built-in Support:** Google Translate, DeepL, OpenAI (ChatGPT), Kimi, Zhipu AI, DeepSeek, and Alibaba Cloud (Qwen).
    *   **Free Option:** Comes with LibreTranslate built-in (no key required).
*   **YouTube Dual Subtitles:** Automatically displays bilingual subtitles on YouTube videos.
*   **Smart Export:**
    *   **Save as Image:** Capture the entire article (original or bilingual) as a long screenshot.
    *   **Export to PDF:** Save articles as clean, paginated PDFs for offline reading.
*   **Highly Customizable:** Adjust font size, color, line spacing, and translation style (Bilingual / Replace / Hover).

**What's New in v1.9.8:**
*   Added OpenRouter support: Access a vast library of AI models.
*   Added Custom Service support: Configure your own API endpoints (OpenAI/Anthropic format).
*   Fixed screenshot and PDF export issues on sites with cross-origin images.
*   Improved settings UI and error handling.

**[中文]**

ZDFTranslate (知道风翻译) 是一款专为高效阅读外语内容设计的浏览器插件。它能在保留原文排版的基础上，智能插入译文，提供流畅的双语对照阅读体验。

**核心功能：**

*   **双语对照阅读：** 智能识别网页段落，在原文下方显示译文，不破坏页面布局。
*   **多翻译引擎支持：**
    *   **OpenRouter (新增)：** 支持接入 OpenRouter 平台，一键使用 GPT-4o, Claude 3.5, Gemini Pro 等数百种顶级模型。
    *   **自定义服务 (新增)：** 支持添加任意兼容 OpenAI 或 Anthropic 格式的 API 接口（如 LocalAI, vLLM 或企业私有部署）。
    *   **主流模型支持：** 原生支持 OpenAI (ChatGPT), Kimi (月之暗面), 智谱清言 (GLM), DeepSeek, 阿里云百炼 (通义千问), Google 翻译, DeepL。
    *   **免费可用：** 内置 LibreTranslate 免费翻译服务，开箱即用。
*   **YouTube 双语字幕：** 观看 YouTube 视频时自动显示双语字幕，学习外语更轻松。
*   **智能导出：**
    *   **长截图：** 将整篇文章（原文或双语）生成高清长图。
    *   **PDF 导出：** 智能分页，将文章导出为整洁的 PDF 文档。
*   **高度个性化：** 可自由调整译文颜色、字号、行间距，支持高亮背景等样式。

**v1.9.8 版本更新：**
*   新增 OpenRouter 支持，聚合数百种 AI 模型。
*   新增自定义翻译服务功能，支持配置任意 OpenAI/Anthropic 兼容接口。
*   修复了部分网站因跨域图片导致的截图和 PDF 导出失败问题。
*   优化了设置页面交互和错误提示。
