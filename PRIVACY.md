# ZDFTranslate 隐私政策

最后更新日期：2026年2月3日

## 简介

ZDFTranslate（"我们"、"本插件"）重视您的隐私。本隐私政策说明我们在您使用 Chrome 浏览器扩展时如何处理信息。

## 信息收集

本插件 **不会收集、存储或传输** 以下信息：
- 您的浏览历史
- 访问的网页内容
- 个人身份信息
- 任何用户行为数据

## 本地存储

本插件仅在您的本地浏览器中存储以下设置：
- 翻译偏好设置（目标语言、显示模式等）
- API 密钥（用于访问第三方翻译服务）
- 排除网站列表

这些数据仅保存在您的设备上，我们不会访问或收集这些信息。

## 第三方服务

当您使用翻译功能时，页面内容会发送到您选择的第三方翻译服务：

| 服务 | 隐私政策 |
|------|----------|
| Google Translate | https://policies.google.com/privacy |
| DeepL | https://www.deepl.com/privacy |
| OpenAI | https://openai.com/privacy |

请注意，使用这些服务时，您的数据将受其各自的隐私政策约束。

## 权限说明

本插件请求的权限仅用于提供核心功能：

- **storage**：保存用户设置
- **activeTab**：在当前页面执行翻译
- **scripting**：注入翻译脚本
- **host_permissions**：在所有网站运行翻译

## 数据安全

- 所有 API 密钥均使用 Chrome 的安全存储机制保存
- 翻译请求直接发送到第三方服务，不经过我们的服务器
- 本插件不包含任何追踪代码或分析工具

## 变更

我们可能会不时更新本隐私政策。任何变更将在本页面发布。
