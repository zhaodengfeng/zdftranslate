# Privacy Policy — ZDFTranslate

**Last updated:** February 17, 2026

## Data Collection

ZDFTranslate does **not** collect, store, or transmit any personal data. No analytics, no tracking, no telemetry.

## Local Storage

The extension stores the following data **locally in your browser only**:

- Translation preferences (target language, display mode, etc.)
- API keys for third-party translation services
- Excluded website list

This data never leaves your device.

## Third-Party Services

When you use the translation feature, page content is sent to the translation provider you selected. Each provider has its own privacy policy:

| Provider | Privacy Policy |
|----------|---------------|
| Google Translate | https://policies.google.com/privacy |
| DeepL | https://www.deepl.com/privacy |
| OpenAI | https://openai.com/privacy |
| Kimi (Moonshot AI) | https://platform.moonshot.cn/docs/privacy |
| Zhipu AI (GLM) | https://open.bigmodel.cn/dev/howuse/privacypolicy |
| DeepSeek | https://www.deepseek.com/privacy |

## Permissions

All requested permissions are used solely for core functionality:

- **storage** — Save user settings
- **activeTab** — Execute translation on the current page
- **scripting** — Inject translation scripts
- **contextMenus** — Right-click translation menu
- **host_permissions** — Run translation on all websites

## Security

- API keys are stored via Chrome's secure storage API
- Translation requests go directly to the selected provider — no intermediary server
- No tracking code or analytics of any kind
