# Chrome Web Store 上架指南

## 1. 准备材料

### 必需
- ✅ 插件文件（已打包为 zip）
- ✅ 开发者账号（已注册并支付 $5）
- ✅ 图标（16x16, 48x48, 128x128）
- ✅ 截图（至少 1 张，推荐 1280x800 或 640x400）
- ✅ 简短描述（最多 132 个字符）
- ✅ 详细描述

### 推荐
- 宣传图片（1400x560）
- 小型宣传图片（440x280）
- 屏幕录制/视频（可选）

## 2. 创建商品详情

访问 [Chrome Web Store 开发者后台](https://chrome.google.com/webstore/devconsole)

### 填写信息

**商店详情页:**
- 语言：简体中文 / English
- 商品详情名称：ZDFTranslate - 沉浸式双语翻译
- 简短描述：智能双语对照翻译，让阅读外文网页如读母语般流畅
- 详细描述：
```
ZDFTranslate 是一款智能双语对照翻译插件，帮助您轻松阅读外文网页。

主要功能：
🌐 智能识别页面主要内容
📖 双语对照显示，便于学习
🎯 多种显示模式可选
⚡ 支持 Google / DeepL / OpenAI 翻译
🎨 自定义译文样式
🚫 可排除特定网站

使用方式：
1. 安装插件后，访问任意外文网页
2. 点击插件图标开启翻译
3. 在设置中配置您偏好的翻译服务

隐私说明：
本插件仅在用户主动开启时进行翻译，不会收集或上传任何浏览数据。翻译内容直接发送给您选择的翻译服务提供商（Google/DeepL/OpenAI）。
```

### 图标和截图

**图标：** 使用 icons/ 目录下的 PNG 文件

**截图（准备 3-5 张）：**
1. 英文网页双语翻译效果
2. 弹出面板设置界面
3. 选项页面配置界面
4. 不同显示模式对比

截图尺寸：1280x800 或 640x400（推荐 1280x800）

## 3. 定价和分发

- **定价：** 免费
- **付款方式：** 无需设置
- ** Visibility：** 公开

## 4. 隐私权限说明

需要填写的权限说明：

| 权限 | 说明 |
|------|------|
| storage | 用于保存用户的翻译设置和偏好 |
| activeTab | 用于在当前标签页注入翻译脚本 |
| scripting | 用于执行页面内容翻译 |
| host_permissions | 用于在所有网站运行翻译功能 |

**隐私政策 URL：** 可以链接到 GitHub 仓库的 PRIVACY.md

## 5. 提交审核

1. 点击「提交审核」按钮
2. 确认所有必填项已填写
3. 等待审核（通常 1-3 个工作日）

## 6. 审核状态跟踪

在开发者后台可以查看：
- 🟡 审核中
- 🟢 已通过
- 🔴 被拒绝（会附带修改意见）

## 7. 发布后

审核通过后：
- 插件会立即上架或按设置的发布时间上架
- 用户可以通过搜索找到插件
- 可以通过链接直接分享：https://chrome.google.com/webstore/detail/[插件ID]

## 8. 更新版本

1. 修改 manifest.json 中的 version 字段（如 1.0.0 → 1.0.1）
2. 重新打包
3. 在开发者后台点击「更新」
4. 上传新的 zip 文件
5. 提交审核

## 常见问题

**Q: 审核被拒绝了怎么办？**
A: 仔细阅读拒绝原因，通常是权限说明不够详细或隐私政策缺失。修改后重新提交。

**Q: 可以加快审核吗？**
A: 一般不行。但如果修复了安全漏洞可以在提交时备注。

**Q: 插件 ID 在哪里看？**
A: 第一次上传后，在「商品详情」页面可以看到公开的插件 ID。

**Q: 如何设置推广图片？**
A: 在「商品详情」→「宣传图片」上传 1400x560 和 440x280 的图片。
